Title : Online Tutor Booking System
A website where students can find a tutor and book a session.
Built with Flask, HTML, and CSS.




How to Run

1. Install Flask:
   pip install flask
   

2. Go to the project folder:
    cd FINAL_OBS
   
3. Run the app:
    python app.py
   

4. Open your browser and go to:
    http://127.0.0.1:5000
   
Tech Used
- Python / Flask
- HTML / CSS
- Jinja2
